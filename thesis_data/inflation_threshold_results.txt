pi_low(S) are both decreasing in S (stablecoin ecosystem size).

Evidence assessment:

  Test 1 (Logit): inflation coeff = -0.0256 (p = 0.2028)

  Test 2 (Piecewise): optimal breakpoint at 2% inflation
  Test 3 (Interaction): d = 0.0005 (p = 0.9582)
    => Stablecoin ecosystem amplifies inflation effect (SUPPORTS prediction)
  Test 4 (Time-varying): Spearman rho(year, threshold) = -0.4000
    => Threshold declining over time (SUPPORTS prediction)
  Test 5 (Quintiles): Spearman rho = -0.1000 (p = 0.8729)

Note: This is a cross-country panel with ~18 overlapping countries
and 5 years (2020-2024). Small sample sizes limit statistical power.
Results should be interpreted as suggestive, not definitive.
